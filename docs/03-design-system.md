# Design System

## Purpose

This document defines the visual language, design principles, layout system, typography, spacing, color palette, components, animations, and accessibility standards for the Ignite The Soul website.

It translates the brand identity into a practical system that developers can consistently implement throughout the project.

Every screen should feel cohesive, timeless, calm, and intentional.

---

# Design Philosophy

The website should not resemble a traditional business website.

Instead, it should feel like a premium editorial wellness magazine.

Think:

- Editorial
- Spacious
- Timeless
- Elegant
- Organic
- Human
- Calm
- Intentional

Luxury should come from simplicity rather than decoration.

---

# Design Principles

## Whitespace is a Design Element

Empty space is intentional.

Allow layouts to breathe.

Never overcrowd sections.

---

## Typography Leads

Typography should carry much of the visual hierarchy.

Large editorial headlines.

Comfortable reading width.

Generous line spacing.

---

## Photography Over Decoration

Real photography creates emotion.

Decorative graphics should be minimal.

---

## Calm Before Interaction

Interfaces should never compete for attention.

Every interaction should feel natural.

---

## Consistency Creates Trust

Every spacing decision, border radius, button, animation, and typography choice should follow the same system.

# Accessibility

Minimum WCAG AA.

Keyboard accessible.

Visible focus states.

ARIA labels where required.

Reduced motion support.

Alt text for every meaningful image.

Readable font sizes.

Touch targets

Minimum 44px.

---

# Responsive Design

Design mobile first.

Every layout should adapt gracefully.

Desktop should feel expansive.

Mobile should feel intimate.

Never hide important content on mobile.

# Final Principle

The website should never feel designed.

It should feel experienced.

Visitors should leave remembering how peaceful the website felt—not how many animations or components it contained.
